# 🌱 NurtureLand

### Connecting People with Nature Through Tree Plantation

NurtureLand is a web-based platform that promotes environmental conservation by enabling users to request free saplings, donate saplings, and participate in plantation drives. The platform aims to increase greenery, encourage community participation, and create awareness about sustainable environmental practices.

---

## 📖 About the Project

NurtureLand serves as a bridge between individuals, nurseries, NGOs, and environmental organizations. Users can easily request saplings, contribute donations, and join plantation events through a simple and user-friendly interface.

The platform helps promote afforestation and environmental awareness while maintaining records of requests, donations, and participation activities.

---

## ✨ Features

* 🌿 Request Free Saplings
* 🎁 Donate Saplings
* 📅 Participate in Plantation Drives
* 👤 User Registration and Login
* 🏢 Nursery and Partner Integration
* 📢 Environmental Awareness Content
* 📊 Request and Donation Management
* 📝 Event Management System
* 🔍 Track User Participation

## 🚀 Installation

### 1. Clone Repository

```bash
git clone https://github.com/cd nurture
```

### 2. Move Project to XAMPP

Copy the project folder to:

```text
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start:

* Apache
* MySQL

### 4. Import Database

1. Open phpMyAdmin
2. Create a database named:

```text
nurture
```

3. Import:

```text
DB.sql
```

### 5. Run Project

Open your browser:

```text
http://localhost/nurture
```
